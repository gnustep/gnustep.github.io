The Base Tools are command line utilities that come with the GNUstep base distribution.

autogsdoc
Parses Objective-C source to autogenerate GSDoc, and converts GSDoc to HTML.
cvtenc
Converts text files between unicode and other encodings.
defaults
Supports viewing/modification of GNUstep user defaults database.
gdnc
GNUstep distributed notification center.
gdomap
GNUstep distributed object mapper.
pldes
Converts a binary serialized property list (class instance) to a text representation.
plmerge
Merges text property lists into a single property list.
plparse
Checks that each file contains a valid text representation of a property list.
plser
Converts a text representation of a property list to a binary serialized representation..
sfparse
Can verify strings-file format, and convert strings files between different encodings.
xmlparse
A simple XML format verifier and DTD validator.
